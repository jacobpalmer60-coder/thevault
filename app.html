Methodology

Scoring detection reads rec and bonus_rec_te from league settings and maps to a single PPG column (e.g., Avg_PPR_0.5_TEP). That column is used for all positions. Team Prod = optimal starter PPG sum. Dynasty Value = Σ (PPG × Age Factor). Age factors from AGE_CSV; fall back to 1.0. Pick values come from AGE_CSV Early/Mid/Late tiers; tiers derived from first-pass Prod (top3 late, middle4 mid, bottom3 early). Pick years start at 2027 (2026 forced forward). Ownership applied from traded_picks. WindowRaw = 0.5·ValueZ + 0.3·PicksZ + 0.2·(-AgeZ). Percentiles are true ranks: (rank-1)/(N-1)·100, so best = 100%. Archetypes are 3×3 (Prod × Window). Labels use “Window”, never “Win”.
