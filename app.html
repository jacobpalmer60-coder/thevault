Dynasty League Overview

Methodology Show methodology ▼

Production = distance from league average PPG. We compute league mean and standard deviation of optimal starting lineup PPG. Tiers are set by standard deviation bands: Elite ≥ +1.25σ, Strong +0.5 to +1.25σ, Average ±0.5σ, Weak -1.25 to -0.5σ, Rebuild ≤ -1.25σ. This shows how far you are from average, not your rank.

Longevity = future outlook index. Each team gets three indexes where 100 = league average: Value Index = team value / avg value × 100, Picks Index = team picks / avg picks × 100, Youth Index = avg age / team age × 100. Longevity Score = 0.7×Value + 0.2×Picks + 0.1×Youth. 100 = league average outlook. Longevity Score is shown as % above/below average. Tiers: Elite ≥+10%, Strong +5 to +10%, Average ±5%, Weak -5 to -10%, Rebuild <-10%.

League overview focuses on distance to average for Production. Window remains percentile-based for now and will be migrated to distance bands next.

Archetypes

Elite Contender – Production Elite/Strong + Longevity Elite/Strong. Win now and built to last.
Aging Contender – Production Elite/Strong + Longevity Weak/Rebuild. Score now, window closing.
Contender – Production Elite/Strong + Longevity Average. Playoff caliber now, neutral future.
Young Riser / Ascending – Production Strong/Average + Longevity Elite. Trending up.
Pick-Rich Rebuilder – Production Weak/Rebuild + Longevity Elite/Strong. Future focused.
Treadmill – Production Average + Longevity Weak/Rebuild. Boom/bust.
Stripped Rebuilder – Production Weak/Rebuild + Longevity Weak/Rebuild. Full teardown needed.
Rebuilding – Production Weak/Rebuild + Longevity Average. Needs direction.
Balanced Core – Production Average + Longevity Average. No extremes.
