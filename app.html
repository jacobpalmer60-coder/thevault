Methodology

Production: Best-ball optimal starters from roster using PPR PPG. TEs use Avg_PPR_0.5_TEP from the PPG sheet; all other positions use Avg_PPR.

Window Score: 0.5× roster value z + 0.3× picks z + 0.2× (−age z). Percentiles derived across league.

Picks: Next 3 seasons of draft capital, adjusted for trades. Values: R1=100, R2=40, R3=20, R4=10, R5=5.

Archetypes: 3×3 grid of Production % vs Window % (Low <33, Med 33-66, High ≥66).
