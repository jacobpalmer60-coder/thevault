Methodology

Prod % = percentile of optimal-starter PPG using your PPG CSV. Window % = percentile of Window Raw = 0.5·ValueZ + 0.3·PicksZ + 0.2·(-AgeZ). Values from Age CSV. Picks valued from Age CSV pick rows (Early/Mid/Late) with 15% yearly discount, tiered by original owner's PPG tier (Top 3 Late / Mid 4 Mid / Bottom 3 Early). No 2026 picks are shown (already drafted).

Archetypes: 9-zone grid on Prod % vs Window %. Elite Contender, Win-Now Core, Aging Contender, Rising Core, Balanced Build, Stuck in Middle, Full Rebuild, Re-tooling, Tanking.

Built single-file. Data via Sleeper API + Google Sheets CSV. No external libraries.
