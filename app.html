Dynasty League Overview

Methodology

Production %

How good your best possible starting lineup is right now. We build each team’s optimal starters for your league settings and rank them by projected PPR points per game using Avg_PPR_0.5_TEP. 90% means you out-score 90% of the league today.

Window %

Long-term outlook. Composite percentile of 0.5×valueZ + 0.3×picksZ + 0.2×(-ageZ). Value = sum of dynasty values from Age CSV. Picks = sum of future pick values. Age = average roster age.

Pick Values

Year-specific values pulled directly from your Age CSV. A 2027 1st Late and 2028 1st Late have different values in your sheet and are treated separately. Tier of each pick is based on first-pass PPG: top 3 = Late, middle 4 = Mid, bottom 3 = Early.

Playoff-Aware Thresholds

6 playoff teams → contender Production floor = 40th percentile. Window thresholds stay 30 / 70. Elite remains Production ≥70 & Window ≥70.

Archetypes

  • Elite Contender: Production ≥70, Window ≥70, age ≤26.5. Strong now and built to last.
  • Win-Now Fringe: Production ≥40, Window 30-70, age ≥26. Playoff-caliber now with decent future.
  • Aging Contender: Production ≥40, Window <30, age ≥27.5, picksPct ≤30. Win now but window is closing.
  • Young Riser / Ascending: Production 30-70, Window ≥70, age ≤25.5. Elite future, trending up.
  • Balanced Core: No extremes. Solid across the board.
  • Volatile / High-Variance: Production 30-70, Window <30. Decent now, shaky foundation.
  • Stuck in the Middle / Purgatory: Production <40, Window 30-70. Below playoff line now, middling future.
  • Pick-Rich Rebuilder: Production <40, Window ≥70, picksPct ≥70. Bad now, loaded for later.
  • Stripped Rebuilder: Production <40, Window <30. Bottom in both.
